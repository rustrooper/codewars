Надо написать функцию-банкомат. На вход она получает сумму, которую ты хочешь получить и валюту. На выходе должна отдать массив значений формата [номинал, кол-во купюр]. Функция должна пытаться выдать введенную сумму максимально большими купюрами.

Если выведенную сумму нельзя выдать имеющимися купюрами, то надо бросать ошибку.

Пример:

Вход: 15000, RUR

Выход: [[5000, 3]]

Пусть валюты будут рубль, евро, доллар и какая-нибудь, где есть неудобные номиналы
